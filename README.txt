RML Suivi Chantier V10 PWA PRO

Fichiers inclus :
- index.html
- manifest.json
- service-worker.js
- apple-touch-icon.png

Installation iPhone :
1. Ouvrir index.html depuis Safari ou depuis GitHub Pages.
2. Appuyer sur Partager.
3. Choisir “Ajouter à l’écran d’accueil”.
4. Supprimer l’ancien raccourci si l’ancienne icône reste affichée.

Important :
Le service worker fonctionne pleinement quand l’application est hébergée en HTTPS, par exemple avec GitHub Pages.
En ouverture locale depuis Fichiers iPhone, l’application fonctionne, mais le cache PWA peut être limité par iOS.
